# Del CIE-11 al apoyo socioeducativo

Microinfografía web interactiva para una intervención de foro en la asignatura **Salud, Dependencia y Vulnerabilidad Social**.

## Cómo abrir en local

Abre `index.html` directamente en el navegador. No requiere instalación ni servidor.

## Cómo subir a GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube todos los archivos y carpetas de este paquete.
3. Entra en **Settings > Pages**.
4. En **Build and deployment**, selecciona **Deploy from a branch**.
5. Selecciona la rama `main` y la carpeta `/root`.
6. Guarda los cambios. GitHub generará una URL pública.

## Estructura

- `index.html`
- `css/app.css`
- `js/app.js`
- `assets/`
- `.nojekyll`

## Uso académico

La herramienta no diagnostica ni sustituye el navegador oficial de la CIE-11. Su finalidad es traducir categorías sanitarias orientativas en necesidades sociales, apoyos socioeducativos, coordinación y seguimiento.
